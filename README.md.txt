# Employee Management System (SQL Project)

## 📌 Project Overview
This is a SQL-based Employee Management System project.

The project includes:
- Employees
- Departments
- Projects
- Salaries
- Login Tracking

This project demonstrates:
- SQL Joins
- Aggregate Functions
- Subqueries
- Window Functions
- CTEs
- Real Interview Queries

---

## 🛠 Technologies Used
- MySQL
- MySQL Workbench

---

## 📂 Database Tables
1. employees
2. departments
3. projects
4. salaries
5. logins

---

## 🚀 Features
- Employee Reports
- Department Analytics
- Top Performers
- Login Tracking
- Consecutive Login Detection
- Duplicate Salary Detection

---

## 📊 Sample Queries
- Top 2 employees per department
- Employees with above average salary
- Employees who never logged in
- Consecutive login users
